Layer Management

Prerequisites: Tutorial 01 complete · Concepts: FeatureLayer, GraphicsLayer, renderers

Overview

Layers are how you get your data onto the map. The ArcGIS SDK has many layer types — this tutorial focuses on the two you will use most: FeatureLayer for querying and displaying server-hosted feature data, and GraphicsLayer for adding your own custom graphics programmatically.

Layer types — quick reference

The three you'll use most

FeatureLayer — connects to a hosted feature service URL and handles querying, rendering, and caching automatically. GraphicsLayer — a blank canvas for adding your own Graphic objects. TileLayer — displays pre-rendered map tile images, useful for basemap context layers.

Adding a FeatureLayer

1

Require the module and point to a feature service

A FeatureLayer is created by pointing to the URL of an ArcGIS feature service. Esri hosts thousands of public datasets on ArcGIS Online. This example uses a public US earthquake dataset:

index.html 
require([
  "esri/Map",
  "esri/views/MapView",
  "esri/layers/FeatureLayer"  // add this
], function(Map, MapView, FeatureLayer) {

  const map = new Map({ basemap: "dark-gray-vector" });

  const view = new MapView({
    container: "viewDiv",
    map: map,
    zoom: 4,
    center: [-98, 38]  // continental US
  });

  const earthquakeLayer = new FeatureLayer({
    url: "https://services.arcgis.com/Do88DoK2xjbc0LCh/arcgis/rest/services/Earthquake_Data/FeatureServer/0",
    title: "Earthquakes"
  });

  map.layers.add(earthquakeLayer);

});

The map.layers collection manages all layers. add() appends the layer to the top of the stack.

Adding a GraphicsLayer

2

Create a layer and add custom graphics

A GraphicsLayer lets you draw anything on the map — points, lines, polygons — defined in your own JavaScript code. Each item is a Graphic with a geometry and a symbol:

index.html 
require([
  "esri/Map",
  "esri/views/MapView",
  "esri/layers/GraphicsLayer",
  "esri/Graphic"
], function(Map, MapView, GraphicsLayer, Graphic) {

  const map = new Map({ basemap: "streets-navigation-vector" });
  const view = new MapView({ container: "viewDiv", map, zoom: 13, center: [-122.4194, 37.7749] });

  // Create the layer
  const graphicsLayer = new GraphicsLayer();
  map.layers.add(graphicsLayer);

  // Define a point geometry
  const point = {
    type: "point",
    longitude: -122.4194,
    latitude: 37.7749
  };

  // Define a symbol (how the point looks)
  const markerSymbol = {
    type: "simple-marker",
    color: [226, 119, 40],   // orange
    outline: { color: [255, 255, 255], width: 2 }
  };

  // Combine into a Graphic and add to the layer
  const pointGraphic = new Graphic({
    geometry: point,
    symbol: markerSymbol,
    attributes: { name: "San Francisco" },
    popupTemplate: { title: "{name}", content: "You placed a pin here." }
  });

  graphicsLayer.graphics.add(pointGraphic);

});

Toggle layer visibility

Every layer has a visible property. Toggle it to show or hide layers without removing them from the map:

snippet 
// Hide a layer
earthquakeLayer.visible = false;

// Show it again
earthquakeLayer.visible = true;

// Toggle based on a checkbox
document.getElementById("toggle").addEventListener("change", (e) => {
  earthquakeLayer.visible = e.target.checked;
});

Styling with renderers

Renderers control how a FeatureLayer looks. A SimpleRenderer applies a single symbol to all features. A ClassBreaksRenderer applies different symbols based on an attribute value:

snippet — SimpleRenderer 
const renderer = {
  type: "simple",
  symbol: {
    type: "simple-marker",
    size: "8px",
    color: "#00b4a0",
    outline: { color: "white", width: 1 }
  }
};

earthquakeLayer.renderer = renderer;

Removing layers

Remove a specific layer or clear all layers from the map:

snippet 
// Remove a specific layer
map.layers.remove(earthquakeLayer);

// Remove all layers
map.layers.removeAll();

// Remove layer by index (0 = bottom of stack)
map.layers.removeAt(0);

Full example

index.html — complete layer management example 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Layer Management</title>
  <link rel="stylesheet" href="https://js.arcgis.com/4.29/esri/themes/light/main.css">
  <script src="https://js.arcgis.com/4.29/"></script>
  <style>
    html, body, #viewDiv { height: 100%; margin: 0; }
    #controls {
      position: absolute; top: 16px; left: 16px; z-index: 10;
      background: white; padding: 12px 16px; border-radius: 8px;
      box-shadow: 0 2px 8px rgba(0,0,0,0.15);
    }
  </style>
</head>
<body>
  <div id="controls">
    <label>
      <input type="checkbox" id="toggle" checked>
      Show earthquake layer
    </label>
  </div>
  <div id="viewDiv"></div>
  <script>
    require([
      "esri/Map", "esri/views/MapView", "esri/layers/FeatureLayer"
    ], function(Map, MapView, FeatureLayer) {
      const map = new Map({ basemap: "dark-gray-vector" });
      const view = new MapView({ container: "viewDiv", map, zoom: 4, center: [-98, 38] });
      const layer = new FeatureLayer({
        url: "https://services.arcgis.com/Do88DoK2xjbc0LCh/arcgis/rest/services/Earthquake_Data/FeatureServer/0"
      });
      map.layers.add(layer);
      document.getElementById("toggle").addEventListener("change", e => {
        layer.visible = e.target.checked;
      });
    });
  </script>
</body>
</html>
