Tutorial 01 — Beginner · ~15 min

Map Rendering

Prerequisites: basic JavaScript, ArcGIS API key · No build step required

Overview

This tutorial walks you through initialising your first ArcGIS map in the browser — from a blank HTML file to a fully interactive map with a chosen basemap and camera position. By the end you will understand the relationship between the Map class and the MapView class, which is the conceptual foundation for everything else in the SDK.

Note

The ArcGIS Maps SDK for JavaScript uses AMD modules loaded via the Esri CDN. You do not need Node.js or a bundler for these tutorials — everything runs in a plain HTML file.

Step 1 — Project setup

1

Create your HTML file

Create a new folder and inside it create a file called index.html. Add the following boilerplate:

index.html 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My First ArcGIS Map</title>
</head>
<body>
  <div id="viewDiv"></div>
</body>
</html>

The viewDiv element is where the map will be rendered. Make it fill the viewport with CSS:

index.html — <head> styles 
<style>
  html, body, #viewDiv {
    height: 100%;
    margin: 0;
    padding: 0;
  }
</style>

Step 2 — Load the SDK

2

Add the Esri CDN links

The SDK is loaded from Esri's CDN. Add these two lines to your <head>:

index.html — <head> 
<link rel="stylesheet"
  href="https://js.arcgis.com/4.29/esri/themes/light/main.css">
<script src="https://js.arcgis.com/4.29/"></script>
Tip

Pin to a specific version (e.g. 4.29) rather than loading latest so your app doesn't break when Esri ships breaking changes.

Step 3 — Create the Map

3

Require the Map module and set a basemap

Add a <script> tag below your viewDiv. Use require() to import the modules you need:

index.html 
<script>
  require([
    "esri/Map",
    "esri/views/MapView"
  ], function(Map, MapView) {

    const map = new Map({
      basemap: "streets-navigation-vector"
    });

  });
</script>

The basemap property accepts any of Esri's named basemap strings. Common options:

  • "streets-navigation-vector" — clean street map (default for web apps)
  • "satellite" — aerial imagery
  • "topo-vector" — topographic
  • "dark-gray-vector" — dark theme
  • "osm" — OpenStreetMap

Step 4 — Initialise the MapView

4

Connect the map to the DOM

The Map object holds data. The MapView renders it. They are separate by design — the same Map can power both a MapView (2D) and a SceneView (3D).

index.html — inside require callback 
const view = new MapView({
  container: "viewDiv",  // matches our div id
  map: map,
  zoom: 13,
  center: [-122.4194, 37.7749]  // [longitude, latitude]
});
Important

ArcGIS uses [longitude, latitude] order — the opposite of many mapping libraries. center: [-122.4194, 37.7749] puts you in San Francisco.

Step 5 — Set camera and extent

5

Fine-tune the viewport

Instead of zoom and center, you can define a precise geographic extent using the extent property:

index.html 
const view = new MapView({
  container: "viewDiv",
  map: map,
  extent: {
    xmin: -122.5,
    ymin: 37.7,
    xmax: -122.35,
    ymax: 37.85,
    spatialReference: { wkid: 4326 }  // WGS84
  }
});

wkid: 4326 is the EPSG code for WGS84 — standard lat/lon coordinates. You'll encounter this constantly in geospatial work.

Full example

Here is the complete file. Open it in your browser — you should see an interactive street map centred on San Francisco.

index.html — complete 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First ArcGIS Map</title>
  <link rel="stylesheet"
    href="https://js.arcgis.com/4.29/esri/themes/light/main.css">
  <script src="https://js.arcgis.com/4.29/"></script>
  <style>
    html, body, #viewDiv { height: 100%; margin: 0; padding: 0; }
  </style>
</head>
<body>
  <div id="viewDiv"></div>
  <script>
    require([
      "esri/Map",
      "esri/views/MapView"
    ], function(Map, MapView) {

      const map = new Map({
        basemap: "streets-navigation-vector"
      });

      const view = new MapView({
        container: "viewDiv",
        map: map,
        zoom: 13,
        center: [-122.4194, 37.7749]
      });

      view.when(function() {
        console.log("Map is ready!");
      });

    });
  </script>
</body>
</html>
