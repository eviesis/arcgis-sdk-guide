Spatial Queries

Prerequisites: Tutorials 01 & 02 complete · Concepts: Query, FeatureLayer.queryFeatures(), async/await

Overview

Spatial queries let you ask geographic questions: "give me all earthquake features within 50km of this point" or "return every city with a population above 500,000." The ArcGIS SDK handles this through the Query object and the FeatureLayer.queryFeatures() method, which returns a Promise resolving to a FeatureSet.

Attribute vs spatial queries

An attribute query filters features by field values (like a SQL WHERE clause). A spatial query filters by geographic relationship — within a polygon, intersecting a line, within a buffer distance. Both use the same Query object; you just set different properties.

The Query object

A Query defines what you're asking for. The most important properties:

  • where — SQL WHERE clause for attribute filtering ("magnitude > 5")
  • geometry — a geometry to filter by spatial relationship
  • spatialRelationship — how to apply the geometry ("intersects", "contains", "within")
  • outFields — which attribute fields to return (["*"] for all)
  • returnGeometry — whether to include geometry in results (needed to display on map)
  • num / start — for pagination

Attribute query — filter by field value

1

Query earthquakes with magnitude above 5

snippet 
require([
  "esri/layers/FeatureLayer",
  "esri/rest/support/Query"
], function(FeatureLayer, Query) {

  const layer = new FeatureLayer({
    url: "https://services.arcgis.com/Do88DoK2xjbc0LCh/arcgis/rest/services/Earthquake_Data/FeatureServer/0"
  });

  const query = new Query({
    where: "magnitude > 5",
    outFields: ["magnitude", "place", "time"],
    returnGeometry: true
  });

  layer.queryFeatures(query).then(function(results) {
    console.log("Features found:", results.features.length);
    results.features.forEach(feature => {
      console.log(feature.attributes.place, feature.attributes.magnitude);
    });
  });

});

queryFeatures() returns a Promise. When it resolves, results.features is an array of Graphic objects — each with .attributes and .geometry.

Geometry query — filter by location

2

Find all earthquakes within a drawn polygon

When a user clicks or draws on the map, you can use that geometry as the spatial filter. Here's a point-with-buffer example using geometryEngine:

snippet — spatial query with buffer 
require([
  "esri/layers/FeatureLayer",
  "esri/rest/support/Query",
  "esri/geometry/geometryEngine",
  "esri/geometry/Point"
], function(FeatureLayer, Query, geometryEngine, Point) {

  // A centre point (San Francisco)
  const center = new Point({
    longitude: -122.4194,
    latitude: 37.7749
  });

  // Create a 500km buffer polygon around the point
  const buffer = geometryEngine.geodesicBuffer(center, 500, "kilometers");

  const query = new Query({
    geometry: buffer,
    spatialRelationship: "intersects",
    outFields: ["*"],
    returnGeometry: true
  });

  layer.queryFeatures(query).then(function(results) {
    console.log(`Found ${results.features.length} earthquakes within 500km`);
  });

});
Tip — geodesic vs planar buffers

Use geodesicBuffer() for real-world distances on a sphere. buffer() (planar) works fine for small areas but distorts at large distances or high latitudes.

Display results on the map

3

Add query results to a GraphicsLayer

Query results are plain Graphic objects — add them directly to a GraphicsLayer to highlight them on the map:

snippet 
const resultsLayer = new GraphicsLayer();
map.layers.add(resultsLayer);

const highlightSymbol = {
  type: "simple-marker",
  color: [255, 80, 80],
  outline: { color: "white", width: 1.5 },
  size: "10px"
};

layer.queryFeatures(query).then(function(results) {
  resultsLayer.graphics.removeAll();  // clear previous results

  results.features.forEach(feature => {
    const graphic = new Graphic({
      geometry: feature.geometry,
      symbol: highlightSymbol,
      attributes: feature.attributes,
      popupTemplate: {
        title: "M{magnitude} — {place}",
        content: "Time: {time}"
      }
    });
    resultsLayer.graphics.add(graphic);
  });
});

Pagination

Feature services return a maximum number of records per request (often 1,000 or 2,000). Use num and start to paginate through large datasets:

snippet — async pagination 
async function getAllFeatures(layer, whereClause) {
  const countQuery = layer.createQuery();
  countQuery.where = whereClause;
  const { count } = await layer.queryFeatureCount(countQuery);

  const pageSize = 1000;
  const pages = Math.ceil(count / pageSize);
  const allFeatures = [];

  for (let i = 0; i < pages; i++) {
    const query = layer.createQuery();
    query.where = whereClause;
    query.outFields = ["*"];
    query.returnGeometry = true;
    query.num = pageSize;
    query.start = i * pageSize;

    const results = await layer.queryFeatures(query);
    allFeatures.push(...results.features);
  }

  return allFeatures;
}

Full interactive example

This example renders a map, runs an attribute query on load, and displays matching features with popups.

spatial-query.html — complete 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Spatial Query Demo</title>
  <link rel="stylesheet" href="https://js.arcgis.com/4.29/esri/themes/dark/main.css">
  <script src="https://js.arcgis.com/4.29/"></script>
  <style>
    html, body, #viewDiv { height: 100%; margin: 0; background: #1a2e3f; }
    #info {
      position: absolute; top: 16px; right: 16px; z-index: 10;
      background: rgba(26,46,63,0.9); color: white;
      padding: 12px 16px; border-radius: 8px; font-family: sans-serif;
      font-size: 14px; min-width: 180px;
    }
  </style>
</head>
<body>
  <div id="info">Loading results...</div>
  <div id="viewDiv"></div>
  <script>
    require([
      "esri/Map", "esri/views/MapView",
      "esri/layers/FeatureLayer", "esri/layers/GraphicsLayer",
      "esri/Graphic", "esri/rest/support/Query"
    ], function(Map, MapView, FeatureLayer, GraphicsLayer, Graphic, Query) {

      const resultsLayer = new GraphicsLayer();
      const quakeLayer = new FeatureLayer({
        url: "https://services.arcgis.com/Do88DoK2xjbc0LCh/arcgis/rest/services/Earthquake_Data/FeatureServer/0",
        visible: false
      });

      const map = new Map({
        basemap: "dark-gray-vector",
        layers: [quakeLayer, resultsLayer]
      });

      const view = new MapView({
        container: "viewDiv", map,
        zoom: 3, center: [-98, 38]
      });

      const query = new Query({
        where: "magnitude >= 6",
        outFields: ["magnitude", "place"],
        returnGeometry: true
      });

      quakeLayer.queryFeatures(query).then(function(results) {
        document.getElementById("info").textContent =
          `M6+ earthquakes: ${results.features.length}`;

        results.features.forEach(f => {
          resultsLayer.graphics.add(new Graphic({
            geometry: f.geometry,
            symbol: { type: "simple-marker", color: [255,80,80], size: "8px" },
            attributes: f.attributes,
            popupTemplate: { title: "M{magnitude}", content: "{place}" }
          }));
        });
      });
    });
  </script>
</body>
</html>
